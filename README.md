# ProductInsightBot

ProductInsightBot is an AI-based system that analyzes and provides insights about watch products using web scraping, data analysis, and natural language processing.

## Demo
https://product-insight-bot.vercel.app/

Demo video:
https://github.com/user-attachments/assets/8e071a75-c3ae-47c8-a42b-98f70a0ab690

## Features

- Web scraping for automated data collection  
- Natural language query understanding  
- Price analysis (affordable vs premium)  
- Rating-based filtering  
- Brand comparison  
- Category insights (Smartwatch, Luxury, Sports, Fashion, Digital, Analog)  

## Technologies Used

- Python  
- Pandas  
- BeautifulSoup4  
- Requests  
- SQLite  
- Regular Expressions
